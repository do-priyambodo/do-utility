XSL Transform

Syntax 
        
xslTransform(xml, xsl, params = {})
Description Transform the specified XML string using the specified XSL string and the specified parameters.
Note: XSL version 1.0 is supported.
Input parameter xml: The input XML string.

xsl: The input XSL string.

params: The input parameters that are passed to the XSL transformation.

  • The parameters are passed as a JSON object with keys representing the parameter name and values representing the parameter value.
  • The parameter values can be of type STRING, INTEGER or BOOLEAN.
    • Example usage: 
    {
  "param1": "value1",
  "param2": 123,
  "param3": true
}

Return type STRING
Output Returns the output of the XSL transformation performed on XML using XSL.
