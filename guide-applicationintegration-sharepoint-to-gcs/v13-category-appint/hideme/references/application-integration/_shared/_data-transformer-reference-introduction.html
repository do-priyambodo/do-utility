This page describes the additional set of predefined Jsonnet functions for the Data Transformer Script task available in {{application_integration_name}}.

Before you begin

To use the following predefined functions, you must import the functions library in your script. Importing the functions library lets you use both the standard Jsonnet functions and the predefined Data Transformer functions.

{{application_integration_name}} supports Jsonnet functions library v0.20.0. For information about the Jsonnet standard functions, see Jsonnet Standard Library.
