**Custom connectors [Preview]**

Application Integration now supports custom connectors. The custom connectors feature (based on the Open API specification) lets you create your own connectors that aren't a part of the standard connectors provided by Integration Connectors. You can use these connectors in your integrations. For more information, see [Custom connectors](/integration-connectors/docs/custom-connectors).
