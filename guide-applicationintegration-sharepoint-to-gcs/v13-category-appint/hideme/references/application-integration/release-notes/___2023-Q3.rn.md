**Connector Event triggers** ([Preview](https://cloud.google.com/products#product-launch-stages))

Application Integration introduces **Connector Event triggers**; specialized triggers that let you invoke an integration based on the [event subscriptions](/integration-connectors/docs/eventsubscription) created in various business applications using [Integration Connectors](/integration-connectors/docs).

The following Connector Event triggers are available in [preview](https://cloud.google.com/products#product-launch-stages):

*  [Jira Cloud trigger](/application-integration/docs/configure-jira-cloud-trigger)
*  [ServiceNow trigger](/application-integration/docs/configure-servicenow-trigger)
*  [Zendesk trigger](/application-integration/docs/configure-zendesk-trigger)

For more information, see [Connector Event triggers](/application-integration/docs/triggers#connector-event-triggers-preview).

{% endreleasenote %}
