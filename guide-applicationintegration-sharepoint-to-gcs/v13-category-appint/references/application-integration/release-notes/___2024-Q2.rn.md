**Loop Metadata variables are changing**

In the <a href="https://cloud.google.com/application-integration/docs/configure-for-each-loop-task">For each loop</a> and <a href="https://cloud.google.com/application-integration/docs/configure-while-loop-task">While loop</a> tasks, there's a `Loop metadata` variable in which you will find duplicate keys for the output variable–for example, `Current Iteration Count` and `current_iteration_count`. We recommend you to use the variables that contain the underscore (`_`) symbol because the other keys are being deprecated. 

For more information, see [Known issue: Duplicate keys in the Loop metadata](/application-integration/docs/known-issue#duplicate_keys_in_the_loop_metadata).

{% endreleasenote %}
