**New canvas view**

In the integration editor layout, you can try the [new canvas view](/application-integration/docs/canvas-view) to create integration flows. This feature is in [preview](https://cloud.google.com/products#product-launch-stages). The canvas view offers the following benefits:

+ Improved responsiveness of the canvas interaction
+ Clearer view of your integration
+ Minimap view
+ Easier to build integrations

{% endreleasenote %}
