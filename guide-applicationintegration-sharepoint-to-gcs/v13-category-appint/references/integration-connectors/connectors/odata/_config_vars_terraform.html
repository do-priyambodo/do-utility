When creating this connection by using Terraform, you must set the following variables in your Terraform configuration file:

Parameter name Data type Required Description
continue_on_error BOOLEAN False Whether or not to continue after encountering an error on a batch request.
cookies STRING False Allows cookies to be manually specified in name=value pairs separated by a semicolon.
custom_url_params STRING False The custom query string to be included in the request.
data_format ENUM False The data format to retrieve data in. Select either ATOM or JSON. Supported values are: ATOM, JSON
enable_atomic_batch_operations BOOLEAN False Whether or not the CData ADO.NET Provider for OData should use atomic batch operations.
include_navigation_parent_columns BOOLEAN False Indicates if navigation parent columns should be included on navigation views.
include_reference_column BOOLEAN False Adds a input only ParentReference column for bulk inserts to properly associate children during a deep insert with the same parent.
navigation_properties_as_views BOOLEAN False A boolean indicating navigation properties should be promoted to full views.
odata_version STRING False The version of OData to use. By default the provider will attempt to autodetect the version.
server_time_zone STRING False The timezone by which the server's Edm.DateTime values are represented. The value of this property will affect how Edm.DateTime filters and results are converted between the server and the client machine.
share_point_ssodomain STRING False The domain of the user when using single sign-on (SSO).
share_point_use_sso BOOLEAN False Whether or not to use single sign-on (SSO) to authenticate to SharePoint Online.
stored_procedures_as_views BOOLEAN False A boolean indicating if we should list stored procedures which return a collection of entities as views.
supports_expand BOOLEAN False Whether you need to specify the base entity's key to query navigation property views.
supports_formulas BOOLEAN False A boolean indicating if the odata service supports server side formulas.
use_client_side_paging BOOLEAN False Whether or not the CData ADO.NET Provider for OData should use client side paging.
use_etags BOOLEAN False Whether or not the OData source uses Etags.
